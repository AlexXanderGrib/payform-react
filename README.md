# Платёжная форма на React + Typescript + Tailwind 

## [Демо](https://payform-react.pages.dev/)

- Адаптированна даже под IPhone 5/SE
- Определяет платёжную систему карты
- Автозаполнение карты
- Правильная клавиатура на телефоне
- Тёмная тема
